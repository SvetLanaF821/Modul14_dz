14.10. Домашнее задание.


Программа выполняет следующие действия:

1. Показывает все слова, которые есть в словаре (изначально там находятся несколько слов на букву "k" английского алфавита).

2. Поиск слова с автодополнением. 

   По нажатию пользователем первой введенной буквы выполнится поиск подходящих слов из словаря. Если нужное слово есть в предложенном списке (не более 5 штук), то он может выбрать его, нажав на нужную клавишу.
Если же нужного слова нет с списке, то пользователь может продолжить вводить слово дальше, т.е. вторую букву.
Далее снова выведется список предложенных слов с учетом введенного префикса (уже двух букв) и возможностью выбора.
Если нужного слова нет в словаре предоставляется возможность выхода в главное меню, нажав пробел.

3. Добавляет слово в словарь.

4. Удаляет слово из словаря.
